# BREZ OS Repository Manifest

## Purpose
Production Next.js business operating system for BREZ: dashboards, integrations, and operational tools.

"Making 60 people operate like 600."

## What Belongs Here
- **Next.js app**: src/app/, src/components/, src/lib/
- **OWL interface**: Voice-first AI assistant UI components
- **Integrations**: Shopify, QuickBooks, Google APIs
- **Operational docs**: docs/rules/, docs/handoff/
- **Business docs**: AP plans, improvement plans, dashboard specs
- **Brand/content**: Documentary crew guides, brand guidelines

## What Does NOT Belong Here
- SEED protocol theory/documentation (→ seed repo, link instead)
- LIVE FREE philosophy (→ owls repo, link instead)
- PersonaPlex/Moshi model code (→ seed repo)
- Abstract mathematical frameworks (→ seed repo)

## Connections
- **seed**: Uses SEED protocol for AI behavior; calls seed/server/app.py for model inference
- **owls**: Applies LIVE FREE philosophy to operational culture

## Key Directories
- `src/app/` - Next.js pages and API routes
- `src/components/owl/` - OWL voice interface components
- `docs/rules/` - Operational rules and governance
- `docs/handoff/` - Project context and architecture
